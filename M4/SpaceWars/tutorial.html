Main Menu

This is the main menu in which you can view all the contents of the game. We have 5 options in which each one does a different thing, which we will explain later.

Options

Option 1

If we go to option 1, we can see the statistics of our planet, which are very necessary when it comes to knowing the resources we have:


  • We can see the level of technology of our defenses and our attack artillery.

  • We can also see the defense units we have.

  • We also have attack units.

  • And finally we have the resources that we currently have to be able to create units.

Option 2

Option 2 will help us create troops, several menus will appear to be able to select the type of troops we want to create, we can choose between defense and attack. Once we have selected the troop we want to create, we can choose how many units we want to create at a cost and if we do not have enough resources, we will get a message that we do not have enough resources. In the images, we can see the options.


Option 3

Option 3, we can improve the technologies:

  1. We can see the level we currently have.
  2. We can see the cost of the upgrade and the total deuterium we currently have to upgrade our troops.

Option 4

In option 4, we can see the battle between the planets and we will get the information about each planet, with its units. We will be able to see what is happening at all times and the battles last 3 minutes.
